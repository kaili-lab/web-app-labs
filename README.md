# web-app-labs

这是一个面向现代应用开发的学习型仓库。

它的目标不是堆积技术名词，而是把零散的前端、后端、架构、产品与 AI 相关知识，整理成一套可以逐层缩放的认知地图。

## 这个仓库重点在做什么
- 先从 high level 视角理解：一个现代应用会遇到哪些问题、为什么会出现、常见方案和代价是什么
- 再把前端、服务端、数据、运行时、协作方式和 AI 能力串联起来，而不是分散成互不相干的知识点
- 最后按需要下钻到具体专题，例如渲染、状态管理、认证、异步并发、缓存、RAG、Agent 等

## 核心方法
- 先搭地图，再看细节
- 先理解问题和压力，再理解实现方案
- 先建立连接关系，再决定往哪个局部深挖
- high level 先行，不等于拒绝底层；而是先知道为什么值得深入
